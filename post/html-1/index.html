HTML Basics

HTML stands for HyperText Markup Language and it is used to structure and present content on the web. Here are some basic HTML tags you should know:

Body

The <body> tag defines the main content of the HTML document. It contains all the content that will be displayed on the web page:

Headings

Headings are used to define the hierarchy of your content:

There are six levels of headings, from <h1> to <h6>. Usually <h1> is for the most important heading and <h6> is the least important.

Link Attributes

Attribute Description
href Specifies the URL of the page the link goes to
target Specifies where to open the linked document
download Specifies that the target will be downloaded when a user clicks on the hyperlink
